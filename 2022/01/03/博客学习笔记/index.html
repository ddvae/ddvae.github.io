博客学习笔记

Posted on 2022-01-03

0.注意标题里边冒号后面需要加空格

1.下载安装node.js(www.nodejs.org) 需要添加到环境变量

2.下载cnpm 命令是：npm install -g cnpm –registry=https://registry.npm.taobao.org

3.下载hexo框架 cnpm install -g hexo-cli(可以用 hexo -v 来验证是否下载成功)

4.创建一个blog文件夹（之后所有的操作都在这个目录下进行）

5.初始化一个博客，命令：hexo init

6.运行hexo服务器 hexo s

7.可以在地址栏输入：localhost:4000来验证是否运行成功

8.创建新博客，命令为：hexo new “博客名”(创建的博客文件放在blog/source/_posts)

9.清理：hexo clean 生成：hexo g 运行：hexo s

本地测试完成，接下来为部署到远端（github）

1.在github上新建一个仓库，名字是owner（自己owner名字）.github.io(必须这样设置)

2.在blog目录下安装git插件，cnpm install –save hexo-deployer-git

3.在_congfig.yml中设置一些参数

​ #Deployment

​ deploy:

​ type: git

​ repo: 创建的仓库地址

​ branch: master

(注意以上三个参数冒号后面都要加空格)

  1. 部署：hexo d(deploy)
  2. 在浏览器访问：输入owner(自己的owner名字).github.io

换主题

git clone github主题地址 themes/主题名字
